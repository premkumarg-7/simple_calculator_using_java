# Java Swing Calculator

This project is a simple calculator implemented using the Java Swing library. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Prerequisites

- Java Development Kit (JDK) installed on your system.
- Basic knowledge of Java programming and Swing concepts.

## Getting Started

Follow these steps to set up and run the calculator application on your local machine:

1. Clone the repository or download the source code files.

2. Open a command prompt or terminal and navigate to the project directory.

3. Compile the source files using the following command:

   ```shell
   javac Calculator.java
   ```

4. Run the application:

   ```shell
   java Calculator
   ```

## Usage

Upon running the application, the calculator window will appear. You can perform the following actions:

- Use the number buttons (0-9) to input values.
- Use the operation buttons (+, -, *, /) to perform arithmetic operations.
- Press the equals (=) button to get the result.
- Use the clear (C) button to reset the calculator.

## Example

To illustrate the usage, let's perform a simple addition operation:

1. Enter the first number by clicking on the respective number buttons.
2. Click the addition (+) button.
3. Enter the second number.
4. Click the equals (=) button to get the result.

The result will be displayed in the calculator window.
